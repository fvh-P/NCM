## できること

+ 名刺情報の作成 (1 アカウントでいくつでも作成可能)

+ 名刺画像の登録 (1 名刺につき表と裏の2枚まで)

+ 閲覧の際の合言葉の設定 (設定していないものは誰でも見れる)

+ 名刺ページへアクセスできる QR コードの自動生成

+ 担当アイドルの設定

+ SNS 投稿時などにおける名刺画像のサムネイル化 (OGP 対応)

+ 他人の名刺を受け取りリストへ追加する
